Blog搭建

基于Hexo搭建的，感觉用github记录人生百态，似乎更符合程序员的尿性
1.install hexo
npm i hexo-cli -g
hexo init
npm install
hexo g;hexo s

2.本地hexo连接github
ssh-keygen -t rsa -C “chundonglinlin@163.com“
cat ~/.ssh/id_rsa.pub
ssh -T git@github.com

3.写文章and发布
hexo new post “article title”
markdown编辑source/_posts/xxxx.md
hexo g
hexo s
上传git： hexo d
